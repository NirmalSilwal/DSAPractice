## 🛠️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/NirmalSilwal/DSAPractice.git
   ```

2. Navigate to the project:

   ```bash
   cd DSAPractice
   ```

3. Compile and run Java files:

   ```bash
   javac src/DSA/FileName.java
   java src/DSA/FileName
   ```
